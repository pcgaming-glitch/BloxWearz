# BloxWearz

A Roblox clothing marketplace built with Next.js 14 and Supabase.

## Features
- User accounts
- Roblox avatar integration
- Product shop
- Reviews
- Support system with automatic emails
- Admin panel
- Release planner
- 2FA login
- Dark/Light mode
- Mobile, desktop and console UI

## Tech Stack
- Next.js 14 (App Router)
- Supabase
- Nodemailer
- JWT Authentication

## Deployment
This project is optimized for Vercel.
